### 생각 정리

#### 기술 스택 리스트
- 회원 관리 : 소셜로그인 ( 구글 )
- ERD 설계 : https://dbdiagram.io
- DataBase : MySQL, JPA or JDBC
- BackEnd : Spring Boot
- FrontEnd : Thymeleaf
- 형상 관리 : Github
- 배포 (미정) : AWS ( EC2, RDS, S3 )
